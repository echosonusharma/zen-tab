# ZenTab

- move between tabs
- tab search (fizzy search)

## Install

```bash
$ npm install
```

## Development

```bash
npm run dev chrome
npm run dev firefox
npm run dev opera
npm run dev edge
```

## Build

```bash
npm run build chrome
npm run build firefox
npm run build opera
npm run build edge
```

## Release

1. Commit all changes
```bash
git add .
git commit -m "your commit message"
```

2. Create and push tag
```bash
git tag vX.Y.Z
git push origin vX.Y.Z
```
