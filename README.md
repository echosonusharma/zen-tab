# ZenTab

## Install

```bash
$ npm install
```

## Development

```bash
npm run dev chrome
npm run dev firefox
npm run dev opera
npm run dev edge
```

## Build

```bash
npm run build chrome
npm run build firefox
npm run build opera
npm run build edge
```
